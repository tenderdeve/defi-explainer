# DeFi Explainer — Custom Agents

## Available Agents

| Agent | File | Purpose | When to Use |
|-------|------|---------|-------------|
| **issue-creator** | `issue-creator.md` | Creates GitHub issues + milestones from GITHUB-ISSUES.md | After plan approval, when setting up project tracking |
| **pr-reviewer** | `pr-reviewer.md` | Reviews PRs against plan, security, DeFi math, billing logic | On every PR before merge |
| **research** | `research.md` | Researches DeFi protocols, APIs, competitors, web3 patterns | When exploring new protocols, evaluating APIs, competitive analysis |
| **ui-design** | `ui-design.md` | Designs UI components, creates mockups, reviews UX | When building components, reviewing layouts, design decisions |

## How to Invoke

Reference these agents by name when working in this project:
- "Use the issue-creator agent to create all GitHub issues"
- "Use the pr-reviewer agent to review PR #5"
- "Use the research agent to investigate Morpho Blue's API"
- "Use the ui-design agent to design the settings page"

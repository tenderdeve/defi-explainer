# Chat Archive

Significant planning and design conversations stored here for context in future sessions.

## Naming Convention

`YYYY-MM-DD-topic.md`

Examples:
- `2026-05-08-initial-planning-session.md`
- `2026-05-10-billing-model-discussion.md`
- `2026-05-15-ui-review-dashboard.md`

## Template

```markdown
# Chat: [Topic]

**Date:** YYYY-MM-DD
**Participants:** [who was involved]
**Related Issues:** #N, #M

## Context
[What prompted this conversation]

## Key Decisions
1. Decision — Reasoning
2. Decision — Reasoning

## Action Items
- [ ] Action 1
- [ ] Action 2

## Notable Insights
[Anything surprising or non-obvious that came up]
```

## What to Log

- Architecture decisions and their reasoning
- Feature scope changes
- Design debates with resolution
- Research findings that changed our approach
- Tradeoff discussions
- User feedback that shaped direction

## What NOT to Log

- Routine code writing sessions
- Bug fixes with obvious solutions
- Simple Q&A

# Architecture Decision Records

Non-obvious decisions recorded here with full context and reasoning.

## Naming Convention

`YYYY-MM-DD-short-title.md`

## Template

```markdown
# ADR: [Decision Title]

**Date:** YYYY-MM-DD
**Status:** Accepted | Superseded | Deprecated
**Related Issue:** #N

## Context
[What problem or question prompted this decision]

## Decision
[What we decided]

## Consequences
[What this means — both positive and negative]

## Alternatives Considered
1. Alternative — Why rejected
2. Alternative — Why rejected
```
